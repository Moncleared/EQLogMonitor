OpenDKP Log Monitor

Due to the way security is built into browsers, developers can't continuously open and read files on a local computer through the browser. In order for OpenDKP to monitor log files and determine what loot has been dropped, we're using Electron app to run a local websocket on your machine. This websocket simply acts as a small server that monitors your log file for any items and sends those item names back to OpenDKP for processing.

All source code for this Electron app is available and if desired can be compiled and built manually for security purposes.
